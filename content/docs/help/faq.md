## What is Scout?

Scout is a web crawling DSL, making it easy for people to write web crawling scripts and get access to powerful web crawling technology.

## Keyboard shortcuts for search?

- focus: `/`
- select: `↓` and `↑`
- open: `Enter`
- close: `Esc`
